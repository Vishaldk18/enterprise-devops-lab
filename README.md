# Enterprise DevOps Lab
 
## Objective
 
Learn DevOps by taking a simple application from local development to a production-ready platform on AWS using industry-standard tools and practices.
 
## Tech Stack
 
Frontend:
- HTML
- CSS
- JavaScript
- Nginx
 
Backend:
- Python Flask
- Gunicorn
 
Database:
- Redis
 
Containerization:
- Docker
- Docker Compose
 
Cloud:
- AWS
 
Orchestration:
- Kubernetes (Amazon EKS)
 
CI/CD:
- Jenkins
- GitHub Actions
 
Infrastructure:
- Terraform
 
Monitoring:
- Prometheus
- Grafana
- CloudWatch
